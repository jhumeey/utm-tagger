UTM Parameters Reference
Detailed breakdown of each parameter and when to use it.
Parameter Required Example Description
Campaign Sourceutm_source Required google Identifies the referrer — a search engine, newsletter, or other source sending traffic to your site.
Campaign Mediumutm_medium Required cpc The marketing channel used — email, cost-per-click, social, banner, or similar.
Campaign Nameutm_campaign Required spring_sale The specific campaign name, slogan, promo code, or product. Used for campaign-level analysis.
Campaign IDutm_id Optional 12345 Used to identify a specific ads campaign. Helpful when managing multiple campaigns in the same tool.
Campaign Termutm_term Optional running+shoes Used for paid search. Identifies the keywords associated with this ad to track which terms convert.
Campaign Contentutm_content Optional banner_a Used for A/B testing and content-targeted ads. Differentiates between links pointing to the same URL.
